Помощь и FAQ

Как начать использовать бота?

1. Перейдите по ссылке на бота @DUMPBest_bot

2. Нажмите /start для активации

3. Выберите интересующие вас пороги сигналов

4. Настройте фильтры под свою торговую стратегию

Какие пороги сигналов лучше выбрать?

5% сигналы: Для активной торговли и скальпинга

7% сигналы: Оптимальный баланс частоты и качества

12% сигналы: Для консервативной торговли крупными суммами

Как работает технический анализ?

Бот анализирует несколько индикаторов:

  • RSI: Определяет перекупленность/перепроданность
  • MACD: Показывает направление тренда
  • Объемы: Подтверждают силу движения
  • Волатильность: Оценивает риски
Бесплатно ли использование бота?

Да! Базовая функциональность бота полностью бесплатна:

  • Получение торговых сигналов
  • Технический анализ
  • Статистика и отслеживание

Премиум функции добавляются по запросу сообщества.

Настройки бота

Пороги сигналов:

  • 5%: Высокая частота, подходит для скальпинга
  • 7%: Сбалансированный режим (рекомендуется)
  • 12%: Консервативный режим для крупных сделок

Фильтры качества:

  • Минимальный RSI: Фильтр по индикатору RSI
  • MACD подтверждение: Требовать подтверждение тренда
  • Объем: Минимальный объем торгов

API интеграция
